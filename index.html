What is Electron?

  • Electron is a framework that allows you to build Desktop applications with Javascript
  • Open source
  • Developed by GitHub
  • Initially developed for GitHub's Atom editor
  • Microsoft, Facebook, Slack, Docker, etc

How does Electron work?

  • Outputs a native client side application, which bundles Chrome
  • Executes your index.js for the backend
  • Opens a specified html file in Chrome for the frontend
  • Usually trivial to port existing web apps
  • Allows you to build for the Desktop using existing skills
  • Gives you access to the user's filesystem, etc

Getting started

  • Initialise your project as usual
npm init
  • Install electron
npm i -D electron@latest
  • Create your index.html
  • Create your index.js
  • Run electron
"start": "electron ."
npm start

  • What if we want to communicate with the backend process?
  • Chrome uses a multi-process architecture
  • Uses Inter Process Communication to allow it's internal processes to communicate
  • Electron uses IPC to communicate between Chrome and Node
  • On the frontend we use ipcRenderer from the electron module
  • Alternatively, spin up your own API and communicate over HTTP
  • Generally not advised - security concern, and can trip antivirus
